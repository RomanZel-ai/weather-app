# WeatherAI

Flutter-приложение прогноза погоды для Android.

## Что умеет

- поиск города;
- текущая погода;
- почасовой прогноз;
- прогноз на несколько дней;
- избранные города;
- погода рядом со мной без GPS-разрешений;
- работа без VPN через wttr.in;
- автоматическая сборка APK через GitHub Actions.

## Сборка APK

1. Открой вкладку **Actions**.
2. Выбери **Build Android APK**.
3. Нажми **Run workflow**.
4. После зелёной сборки скачай artifact **weather-app-release-apk**.
5. Распакуй ZIP и установи `app-release.apk`.
