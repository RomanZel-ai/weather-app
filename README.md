# WeatherAI

Flutter-приложение прогноза погоды.

## Что умеет

- поиск города;
- текущая погода;
- почасовой прогноз на ближайшие 24 часа;
- прогноз на 14 дней;
- избранные города;
- сборка APK через GitHub Actions.

## Как собрать APK

1. Открой вкладку Actions.
2. Выбери Build Android APK.
3. Нажми Run workflow.
4. Скачай artifact weather-app-release-apk.
